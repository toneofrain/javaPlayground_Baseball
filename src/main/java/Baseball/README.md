## 숫자야구게임

### 핵심 로직

1. 1 ~ 9 중 서로 다른 임의의 수 3개 선택
2. 볼 계산
3. 스트라이크 계산


### UI 로직

1. 3개의 숫자를 모두 맞출 때까지 반복
2. 사용자로부터 숫자를 입력 받음 (숫자를 입력해 주세요 : >)
3. 계산한 스트라이크 볼 출력
4. 모두 맞췄을 시 게임 종료 출력
5. 게임을 새로 시작할지 종료할지 입력받음
6. 게임 새로 시작 / 종료
